## 3. Bottle count filter + hide 0-bottle wines by default

**File:** `app/src/lib/data/filter-sort.ts`

Add:
```ts
export type BottleFilter = 'in-stock' | 'single' | 'empty'

export function filterByBottleCount(wines: WineEntry[], filter: BottleFilter): WineEntry[] {
  if (filter === 'single') return wines.filter(w => w.bottles === 1)
  if (filter === 'empty')  return wines.filter(w => w.bottles === 0)
  return wines.filter(w => w.bottles > 0)   // 'in-stock' = default
}
```

**File:** `app/src/lib/data/filter-sort.test.ts`

Add `describe('filterByBottleCount')` covering all three modes.

**File:** `app/src/routes/+page.svelte`

- Add `let bottleFilter: BottleFilter = $state('in-stock')`
- Wrap display chain: `sortWines(filterByBottleCount(...existing chain..., bottleFilter), activeSort)`
- In complex filter panel, add two toggle buttons (mutually exclusive, both reset to `'in-stock'`):
  - **Single bottle** ↔ `'single'`
  - **0 bottles** ↔ `'empty'`
- Show active `bottleFilter` ('single' or 'empty') as a removable pill alongside type/producer/country pills

## 4. Bottle history — data model

**File:** `schema/cellar.schema.json`

Add optional `history` to WineEntry `properties` (not in `required`):
```json
"history": {
  "type": "array",
  "items": {
    "type": "object",
    "required": ["timestamp", "delta", "bottles"],
    "properties": {
      "timestamp": { "type": "string" },
      "delta": { "type": "integer" },
      "bottles": { "type": "integer", "minimum": 0 }
    },
    "additionalProperties": false
  }
}
```

**File:** `app/src/lib/data/types.ts`

```ts
export interface HistoryEntry {
  timestamp: string  // ISO 8601
  delta: number      // actual change after clamping (+1 or -1)
  bottles: number    // count after this change
}
```

Add `history?: HistoryEntry[]` to `WineEntry`.

---

## 5. Bottle history — recording

**File:** `app/src/lib/data/store.ts`

Update `adjustCount`:
- Compute `newBottles = Math.max(0, wine.bottles + delta)`
- If `newBottles === wine.bottles`, return early (no change — already at 0)
- Otherwise append `{ timestamp: new Date().toISOString(), delta: newBottles - wine.bottles, bottles: newBottles }` to `wine.history ?? []`

**File:** `app/src/lib/data/store.test.ts`

Add 4 tests:
- "records a history entry when count increases"
- "records a history entry when count decreases"
- "does not record a history entry when already at 0 and decremented"
- "accumulates multiple history entries in order"
