# v0.9.8 — Rosé wine type

## Goal
Add 'rose' as a fifth wine type throughout the app, and shrink bottle buttons
slightly so all five fit on one line on a standard iPhone.

## Acceptance criteria
- 'rose' is a valid WineType everywhere (schema, types, form, filter bar)
- bottle-rose.png shown in filter bar alongside red/white/sparkling/dessert
- Bottle filter buttons shrink from w-10/w-8 to w-9/w-7 to keep the row on one line

## Changes
- `schema/cellar.schema.json` — add 'rose' to type enum
- `app/src/lib/data/types.ts` — add 'rose' to WineType union
- `app/src/lib/components/WineForm.svelte` — add Rosé option to type select
- `app/src/routes/+page.svelte` — add 'rose' to typeLabels and bottle {#each}; shrink button sizes
- `app/package.json` — bump to 0.9.8
