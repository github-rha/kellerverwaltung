# Plan: v0.9.0 — Visual redesign

## Context

Redesign the UI to draw visual inspiration from haeusler-wein.ch (github.com/github-rha/haeusler-wein):
new accent color (#A62A17, a brick red), warm grey primary text (#575757), wine-tinted structural
borders (rgba(166,42,23,0.2)), and the Turmfalke (kestrel falcon) as the PWA home-screen icon.
Typography stays unchanged (system sans-serif via Tailwind defaults).

---

## 1. Define custom Tailwind theme colors

**File:** `app/src/app.css`

Add an `@theme` block to introduce named wine color utilities:

```css
@import 'tailwindcss';

@theme {
  --color-wine: #A62A17;
  --color-wine-50: #fdf3f2;
  --color-wine-100: #fbe8e6;
}
```

This makes `text-wine`, `bg-wine`, `border-wine`, `bg-wine-50`, `bg-wine-100` etc. available
as standard Tailwind utilities across all Svelte files.

---

## 2. Replace brand colors in all Svelte files

**Files:** all 9 `.svelte` files under `app/src/`

Contextual (not blanket) replacements. Key distinction:
- **Brand red** (`red-800` / `red-50` / `red-100` for active states, buttons, links, badges) → `wine` / `wine-50` / `wine-100`
- **Error red** (`bg-red-50 border-red-200 text-red-700` alert patterns) → **leave unchanged**

| Old class | New class | Notes |
|---|---|---|
| `text-red-800` | `text-wine` | Links, active filter labels |
| `bg-red-800` | `bg-wine` | Filled buttons, active producer/country items |
| `border-red-800` | `border-wine` | Active pill borders |
| `bg-red-50` (brand context) | `bg-wine-50` | Active filter button backgrounds |
| `bg-red-100` (brand context) | `bg-wine-100` | Wine type badge (red wine), active filter pills |
| `text-gray-900` | `text-[#575757]` | All headings and primary body text |

Leave untouched: `text-red-700`, error banner patterns (`bg-red-50 border-red-200 text-red-700`),
`text-amber-*` sync warnings, `text-gray-400/500/700` secondary text.

---

## 3. Update structural border colors

**Files:** `+page.svelte`, `wine/[id]/+page.svelte`, `wine/new/+page.svelte`, `settings/+page.svelte`, `import/+page.svelte`

Replace `border-gray-200` on section-level containers (header bars, filter bars, sync bars) with
`border-[rgba(166,42,23,0.2)]`.

Leave `border-gray-300` (form inputs) and interior card/dropdown `border-gray-200` unchanged.

## 4. Update theme color

**File:** `app/src/app.html`

```html
<meta name="theme-color" content="#A62A17" />
```

**File:** `app/vite.config.ts`

`theme_color: '#A62A17'`

---

## 5. PWA icon — Turmfalke falcon

**Files:** `app/static/icon-192.png`, `app/static/icon-512.png`

1. Download `images/turmfalke.png` from github-rha/haeusler-wein via `gh api`
2. Use ImageMagick (`convert`) to composite the falcon onto a 512×512 `#A62A17` background, centered and scaled to fit
3. Resize the 512px result down to 192×192
4. Save to `app/static/icon-512.png` and `app/static/icon-192.png`
5. If ImageMagick not available, fall back to `sips` + inline Node.js canvas script
