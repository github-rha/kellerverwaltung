# v0.9.4 — Single filter row

## Goal

Merge the two-row filter area (bottle type images on row 1, Filter + Sort on
row 2) into a single flex row.

## Change

`app/src/routes/+page.svelte` — replace the two separate `<div class="flex …">`
wrappers with one:

```html
<div class="flex items-center gap-2">
  <!-- 4 bottle image buttons -->
  <!-- Filter button with dropdown -->
  <!-- Sort button with dropdown -->
</div>
```

The bottle buttons are 40×40 px; the Filter/Sort pills are shorter. `items-center`
keeps them vertically centred on the same baseline.

## Acceptance criteria

- [ ] Bottle image buttons and Filter/Sort buttons are on the same line
- [ ] Dropdowns still open correctly
- [ ] Active-filter chips row still appears below when filters are active
- [ ] `npm run lint`, `npm run format`, `npm run check`, `npm test` all pass
